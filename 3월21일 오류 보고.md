## 🚨 A. 심각한 오류 (Critical Errors)

### 1. Unit 032: 콘텐츠 불일치 (concept ≠ 연습문제)

> [!CAUTION]
> **제목**: "전치사 (시간)" / **개념톡**: at, on, in 시간 전치사 설명
> **그러나** practiceA, practiceB, descriptive, practical 모두 **"부정대명사 one, another, the other"** 문제로 구성되어 있음

- 개념 설명은 시간 전치사(at/on/in)인데, 연습문제는 완전히 다른 문법 주제(부정대명사)
- 학생이 전치사를 공부하러 왔는데 부정대명사 문제를 풀게 되는 심각한 오류
- **별도의 유닛 090도 "전치사 in, on, at (시간)"으로 존재** → 090은 정상적으로 전치사 문제가 들어있음
- Unit 032의 연습문제를 시간 전치사 문제로 교체하거나, 기존 유닛과의 중복 관계를 정리해야 함

### 2. Unit 044: 제목 오류 — "현재완료 현재완료 진행형"

- "현재완료 / 현재완료 진행형" 또는 "현재완료와 현재완료 진행형"이어야 하는데, 구분자 없이 **"현재완료 현재완료 진행형"**으로 표기
- `units.json`과 `044.json` 모두 동일한 오류

### 3. Unit 044 Practical 05번: 불완전한 문제

```
**05.** 밑줄 친 부분 중 어법상 어색한 것을 골라 고쳐 쓰시오.
Amy has learned drawing since she was > 정답:
```
- 문제 문장 자체가 잘려 있고, 정답도 비어 있음 — **풀 수 없는 문제**

### 4. Unit 044 Practical 09번: 불완전한 정답

```
> 정답: Mina has taken a nap on the bench for an hour.,
```
- 2개를 고르라는 문제인데 **정답이 1개만 표시**되어 있음
- "Tom and I have been talking about games for two hours."도 정답인데 누락

### 5. Unit 044 Practical 19번 / 20번: 정답 형식 혼란

```
19. > 정답: moved, didn't have, met, have been
20. > 정답: lost, hasn't found, hasn't used, has, can't take
```
- "[2개]"라고 했는데 정답에 **4~5개의 단어를 나열** → 어떤 것이 "틀린 것"이고 어떤 것이 "고친 것"인지 구분이 안 됨
- 정답 표기를 `has moved → moved` 와 같이 **"틀린 표현 → 올바른 표현"** 형식으로 명확히 해야 함

## ⚠️ B. 유닛 간 중복/불일치 문제

### 6. Unit 011 vs 012: practiceA 내용 거의 동일

| 구분 | Unit 011 (과거형 부정문/의문문) | Unit 012 (일반동사 부정문) |
|---|---|---|
| practiceA 01~05 | don't/doesn't 고르기 | don't/doesn't 고르기 |
| practiceA 06~10 | 틀린 부분 고치기 | 틀린 부분 고치기 |
| practiceA 11~15 | 빈칸 채우기 | 빈칸 채우기 |
| practiceA 16~20 | 단어 배열 | 단어 배열 |

- Unit 011의 practiceA 타이틀은 **"일반동사의 현재/과거 부정문"**이라서 Unit 012 범위까지 포함
- **학생 입장에서 두 유닛이 거의 동일한 연습문제** → 학습 효과 저하
- Unit 011의 practiceA는 과거형(didn't) 중심으로 차별화하거나, 의문문(Did~?) 문제 비중을 늘려야 함

### 7. Unit 011 vs 012: Practical(변형문제)도 대량 중복

- Unit 011 practical 01~08번과 Unit 012 practical 03~08번이 단어만 살짝 바꾼 동일 구조
- 특히 아래 문제쌍은 **거의 동일**:
  - "Kate는 어제 공원에 가지 않았다" vs "Melissa는 어제 학교에 가지 않았다"
  - "Jake는 그의 가방을 가져오지 않았다" (양쪽 동일)
  - "나는 아침을 먹지 않는다" vs "나는 점심을 먹지 않는다"

### 8. Unit 090 practiceB vs practical: 대량 중복

- Unit 090의 practiceB와 practical이 구조와 보기, 심지어 정답까지 거의 동일 (단어만 일부 변경)
- 예: practiceB 01번 "Tom often go hiking" → practical 01번 "Tom often goes swimming"

## 📝 C. 문법/영어 콘텐츠 오류

### 9. Unit 044 practiceA 13번: 정답 애매

```
그들은 5년 동안 이 학교에 다니고 있다.
They _____ to this school for five years.
> 정답 have gone (have been going)
```
- "have gone"은 "갔다(가서 지금 없다)"의 의미 → **"have attended" 또는 "have been going"이 적절**
- 현재완료 "have gone"은 "~에 가 버렸다"와 혼동할 수 있음

### 10. Unit 164 practiceB 13번: 정답 논란

```
Jenny는 뉴욕에서 공부한다.
> 정답: In New York studies Jenny.
```
- 이 문장은 도치 강조가 아니라 **일반적인 문장** → 평서문 "Jenny studies in New York."이 자연스러움
- 도치구문을 연습하는 의도는 알겠지만, **원래 문장 자체가 도치가 필요없는 상황**이라 오해를 줄 수 있음
- 또한 "In New York **does Jenny study**"가 아닌 "In New York **studies Jenny**"는 **장소 부사구 도치로 주어가 명사인 경우**에만 쓰는데, 이 문장이 가장 자연스러운 표현이 맞는지 논란

### 11. Unit 001 practiceB 18번: 문장-한국어 불일치

```
냉장고에 있는 이 과자들은 그녀의 것이니?
Are these cookies in the kitchen ( ) ?
```
- 한국어는 "**냉장고**"인데 영어는 "**kitchen(부엌)**" → 불일치 (refrigerator가 맞음, 또는 한국어를 "부엌"으로 변경)

### 12. Unit 164 practiceB 16번: "Between the two boys stood she" 정답 판단 논란

- 이 문장은 대명사 주어(she)인 경우 도치가 안 되므로 틀린 게 맞지만, 정답이 "Down the stream did the log float"로 주어짐
- 실제로 "Between the two boys stood she"도 비문법적인데, 정답이 다른 문장으로 되어 있어 **정답이 2개일 가능성** — 검토 필요

## 🔧 D. 형식/포맷 불일치

### 13. 문제 형식 통일 안 됨

| 종류 | practiceA | practiceB |
|---|---|---|
| Unit 001 | "보기에서 알맞은 것을 고르시오" | "괄호 안에 알맞은 것을 고르시오" |
| Unit 012 | "보기에서 알맞은 것을 고르시오" | "보기 안에서 알맞은 것을 고르시오" |
| Unit 090 | 2번부터 문제 지시문이 **통째로 누락** (물음표 `?`로만 표시) | 정상 |

- Unit 090 practiceA 02~05번: 지시문 없이 물음표만 있음  
  ```
  **02.** We have a meeting ? Monday.
  > 정답 on
  ```
  → "다음 빈칸에 알맞은 전치사를 쓰시오." 같은 지시문이 필요

### 14. 서술형 문제 타이틀 형식 불일치

- Unit 001: `# Unit 001 인칭대명사중1 (중등) - 서술형문제` (띄어쓰기 없음, "중1" 포함)
- Unit 012: `# Unit 012 일반동사의 부정문중1 (중등) - 서술형문제`
- 다른 단계: `# Unit 012: 일반동사의 부정문 - 연습문제 A` (콜론 포함)
- **서술형 문제만 유독 포맷이 다름** → 통일 필요

### 15. 정답 표기 불일치

| 유닛/단계 | 정답 표기 |
|---|---|
| practiceA/B | `> **정답** His` |
| descriptive | `정답 His` (볼드 없음, `>` 없음) |
| practical | `> 정답: His` (콜론 있음) 또는 `> **정답**: His` |

- 동일 앱 내에서 3가지 이상의 정답 표기 형식이 혼재

## 💡 E. 사용성 개선 제안

### 16. 유닛 중복/겹침 정리 필요

| 유닛 | 제목 |
|---|---|
| 032 | 전치사 (시간) |
| 033 | 전치사 (장소, 위치) |
| 034 | 전치사 (방향, 이동) |
| 090 | 전치사 in, on, at (시간) |
| 091 | 전치사 before, after, for, during, until, by |
| 092 | 전치사 in, on, at (장소) |
| 093~095 | 기타 전치사 |

- 032~034와 090~095가 **동일 문법 주제를 서로 다른 관점에서 다룸** — 학생 입장에서 어떤 유닛을 먼저 해야 하는지 혼란
- 카테고리 또는 난이도 표시를 추가하여 구분 필요

### 17. 유닛 011–013 관계가 복잡

| 유닛 | 제목 | 실제 practiceA 내용 |
|---|---|---|
| 011 | 일반동사의 과거형 (부정문, 의문문) | 현재+과거 부정문 |
| 012 | 일반동사의 부정문 | 현재+과거 부정문 |
| 013 | 일반동사의 의문문 | — |

- 011과 012의 practiceA가 거의 같고, 011의 의문문 내용은 practiceB에만 있음
- 학생이 "부정문을 공부하러" 왔을 때 011을 할지 012를 할지 헷갈림

### 18. practiceB/practical 간 난이도 차이 불명확

- practiceB가 practiceA보다 약간 어려운 정도인데, practical(변형문제)과 큰 차이가 안 느껴지는 유닛이 많음
- practiceB에서 이미 "2개 고르시오" 등 실전형 문제가 나오면, practical과 역할이 겹침

### 19. 설명 없는 오답(오답 해설 부재)

- 모든 단계에서 정답만 표시하고, **왜 다른 선택지가 틀린지 해설이 없음**
- 특히 실전문제(practical)는 학생이 오답 이유를 이해하기 어려움
- 최소 개념톡(concept)에서 설명된 규칙을 정답 옆에 간단히 언급하면 좋음 (예: "doesn't 뒤에는 동사원형")

### 20. 도치구문(164) 난이도 — 중등 대상에 적합한지 검토

- Unit 164 도치구문은 **고등 문법**에 가까운 내용 (Never have I~, Hardly had she~ 등)
- "중1 (중등)"이라는 서술형 타이틀과 맞지 않음
- 전체 앱이 중등 대상이라면, 고급 유닛은 별도로 표시하거나 난이도 레벨을 부여해야 함
